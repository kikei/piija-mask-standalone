# マスク定義

本仕様の動作はユニットテストで保証する。

## 0. 全体ポリシー

- マスク文字は全角の「＊」とする。
- 置換は検出したひとかたまりを常に「＊＊＊＊」に置換し、元の文字数は保持しない。
- 一部だけを残す処理は行わない。結果は不可逆とする。
- 初期の対象外は URL と、顧客ID・会員番号などのラベル付き ID とする。
- 処理順は正規化 → 住所 → 人名 → メール → 電話 → 郵便 → クレジットカード → マイナンバーとする。
- 住所の終端は句読点、改行、括弧閉じ、次の都道府県名、または文末で打ち切る。
- クレジットカード番号は空白やハイフンを除去して 13–19 桁へ整列し、Luhn チェック合格のみ対象とする。
- マイナンバーは 12 桁のみ対象とし、先頭 11 桁に重み 2,3,4,5,6,7,2,3,4,5,6 を右から対応させて合計し、`11 - (合計 mod 11)`（10 と 11 は 0）を検査数字とする。この検査数字が 12 桁目と一致したもののみ対象とする。
- 置換トークンは常に「＊＊＊＊」とし、隣接するマスク同士は連結しない。

## 1. 正規化（前処理）

- Unicode 正規化は NFKC を適用する。
- ハイフン類（`‐ - – — − ー` など）は `-` に統一する。
- ゼロ幅文字や制御文字は除去する。
- 必要に応じて連続空白を 1 つへ正規化する。

## 2. マスク対象と判定規則

### 2.1 メールアドレス

- 検出は正規表現 `[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}` を用いる。
- 検出範囲全体を「＊＊＊＊」に置換する。

### 2.2 電話番号（国内一般形）

- 検出は正規表現 `0\d{1,4}-?\d{1,4}-?\d{4}` を用い、ハイフンの有無を許容する。
- 検出範囲全体を「＊＊＊＊」に置換する。

### 2.3 郵便番号

- 検出は正規表現 `(?:〒\s*)?(\d{3}-?\d{4})` を用いる。先頭の記号 〒 と空白は任意とし、数字部のみをキャプチャする。
- 置換は数字部のキャプチャだけを「＊＊＊＊」に置換する。`〒` が存在する場合は記号を残す。`〒` が無い場合は数字部全体を「＊＊＊＊」に置換する。

### 2.4 クレジットカード番号

- 検出は空白やハイフンを許容して 13–19 桁へ整列し、Luhn チェック合格のみを採用する。
- 検出範囲全体を「＊＊＊＊」に置換する。

### 2.5 マイナンバー（個人番号）

- 検出は数字 12 桁（ハイフンや空白は許容）で、検査数字が一致するもののみを対象とする。
- 検出範囲全体を「＊＊＊＊」に置換する。

### 2.6 名前（人名）

- 形態素解析で `名詞-固有名詞-人名`（姓、名、一般）に該当するトークンを検出する。
- スペースや中黒、連結のゆらぎ（例: `山田 太郎`、`山田・太郎`、`山田太郎`）や、カタカナ 2–4 文字の 2 語連結（例: `ヤマダ タロウ`）を許容する。
- 敬称（さん、様、氏、殿、くん、ちゃん）はブロックに含めない。
- 括弧書きのフリガナは別ブロックとしてマスク対象とし、可逆性を避ける。
- 検出した 1 トークンまたは氏名ブロックを「＊＊＊＊」に置換する。
- 企業名や製品名などの頻出語は小辞書で除外し、誤検出を抑制する。

### 2.7 住所（日本語）

- 方針は都道府県名のみを残し、それ以降（市区町村、町字、丁目から室番、建物名）はすべて「＊＊＊＊」に置換する。
- **都道府県名自体はマスクしない（仕様）。** 「東京都」「大阪府」「北海道」などの都道府県名は出力に残る。
  マスク対象となるのは都道府県の次のトークン以降（市区町村・番地・建物名など）である。
- 検出は都道府県名（47 名称のいずれか）を起点とし、後続の住所語（市、区、郡、町、村、字、丁目、番地、号、地番、建物、室 など）が続く範囲を 1 ブロックとして抽出する。
- 先頭に `〒NNN-NNNN` がある場合は、都道府県名に到達するまでを読み飛ばしてから抽出する。
- 都道府県名が見つからない住所表現（例: `札幌市中央区…`）は当該住所ブロック全体を「＊＊＊＊」に置換する。
- 都道府県名の一覧は次とする。北海道、青森県、岩手県、宮城県、秋田県、山形県、福島県、茨城県、栃木県、群馬県、埼玉県、千葉県、東京都、神奈川県、新潟県、富山県、石川県、福井県、山梨県、長野県、岐阜県、静岡県、愛知県、三重県、滋賀県、京都府、大阪府、兵庫県、奈良県、和歌山県、鳥取県、島根県、岡山県、広島県、山口県、徳島県、香川県、愛媛県、高知県、福岡県、佐賀県、長崎県、熊本県、大分県、宮崎県、鹿児島県、沖縄県。

## 3. 入出力例

- 入力: `山田太郎の電話は090-1234-5678、メールはtaro@example.com`
  出力: `＊＊＊＊の電話は＊＊＊＊、メールは＊＊＊＊`

- 入力: `〒100-0005 東京都千代田区丸の内1-1-1 ○○ビル8F`
  出力: `〒＊＊＊＊ 東京都＊＊＊＊`

- 入力: `カード番号は4111-1111-1111-1111`
  出力: `カード番号は＊＊＊＊`

- 入力: `札幌市中央区北○条西○丁目 12-3-4`
  出力: `＊＊＊＊`

- 入力: `ヤマダ タロウは本日来社します`
  出力: `＊＊＊＊は本日来社します`
